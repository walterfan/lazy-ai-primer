# Summarize Code

Provide a concise summary of code functionality and structure.

## Summary Format

**Component:** [Name]
**Purpose:** [One-line description]
**Key Responsibilities:**
- Responsibility 1
- Responsibility 2
- Responsibility 3

**Main Classes/Interfaces:**
- `ClassName` - Brief description

**Key Methods:**
- `methodName()` - Brief description

**Dependencies:**
- Service dependencies
- External integrations

**Configuration:**
- Required properties (if any)

## Guidelines
- Keep summary concise (2-3 paragraphs max)
- Focus on "what" and "why", not "how"
- Highlight business purpose
- Mention key integration points
