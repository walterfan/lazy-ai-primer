本章节通过 10 个循序渐进的教程，帮助你掌握 LangChain 框架的核心概念和实践技能。
我们将以构建一个 **自媒体内容创作与发布 AI Agent** 为实战案例，贯穿整个学习过程。

学习目标
========
完成本教程后，你将能够：

- 理解 LangChain 的核心组件和设计理念
- 熟练使用 LLM、Prompt Templates、Chains 等基础组件
- 掌握 Memory 和 Agent 的高级用法
- 实现 RAG（检索增强生成）系统
- 构建完整的自媒体内容创作与发布 Agent

实战项目：自媒体 AI Agent
=========================
我们的目标是构建一个能够：

1. 根据热点话题自动生成内容创意
2. 撰写高质量的文章、视频脚本
3. 自动适配不同平台的格式要求
4. 一键发布到多个自媒体平台
