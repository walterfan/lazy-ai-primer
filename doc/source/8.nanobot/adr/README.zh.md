# 架构决策记录（Architecture Decision Records）

本目录存放 nanobot 项目的架构决策记录（ADR）。

## 什么是 ADR？

ADR 记录一个重要的架构决策，包括决策的背景、考虑过的备选方案以及决策带来的影响。

## 模板

新建 ADR 时请以 `template.md` 为起点。

## 命名规范

ADR 按顺序编号：`0001-title.md`、`0002-title.md`，以此类推。
